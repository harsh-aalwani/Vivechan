# Vivechan — AI-Powered Automated Exam Evaluation System

Vivechan is an AI-driven system designed to automate university answer sheet evaluation. It processes student answer sheets at scale, evaluates responses against the question paper, and generates structured results — all within seconds.

Vivechan has been presented as part of the **SSIP GLS Startup Nexus Exhibition**.

---

## 🚀 Features

- Batch upload and process multiple student answer sheets (PDFs)
- AI-powered evaluation with per-question scoring
- Automatic calculation of scores, percentages, and pass/fail status
- Structured student-wise reports
- Downloadable evaluation outputs in PDF and JSON formats
- Organized report storage by academic year

---

## 🎯 Purpose

Vivechan aims to:

- Reduce manual grading effort
- Improve evaluation consistency
- Enable faster result processing for institutions
- Demonstrate practical use of AI in educational workflows
