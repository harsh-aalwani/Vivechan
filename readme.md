# Vivechan — AI-Powered Automated Exam Evaluation System

Vivechan automates university answer sheet evaluation using AI. Upload student PDFs in bulk, and the system scores each answer against the question paper, calculates grades, and generates structured reports — in seconds.

Built as a working pilot for MCA Semester 2.

## Features
- Batch upload and process multiple student PDFs at once
- AI-powered per-question scoring against the question paper
- Student-wise results with scores, percentages, and pass/fail status
- Downloadable PDF and JSON evaluation reports organised by academic year

## Tech Stack
HTML · CSS · JavaScript · Python · Claude API (Anthropic)

## Academic Context
MCA Semester 2 project — practical application of generative AI in educational technology.
